# Welcome to a mod's GitHub Page!
Helpful Commands is a Minecraft Mod that adds different commands to the game. They can help you save lots of time by typing one word instead of the whole "java-formatted code" or long phrases. This mod also adds some new command-related features that are not possible in the original game.
